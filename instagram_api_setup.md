# Instagram Graph API セットアップガイド

---

## 第1フェーズ：土台作り（アカウントの準備と連携）

> APIを使うための「物理的な器」を用意し、相互にヒモ付けるフェーズです。

### 手順

1. **Facebookページの作成**
   - 空っぽのダミーページでOK。これがないとAPIは始まりません。

2. **Instagramをプロアカウントに変更**
   - スマホアプリから「ビジネス」または「クリエイター」アカウントに切り替えます。

3. **FacebookページとInstagramのリンク（連携）**
   - スマホのInstagramアプリから「プロフィールを編集」→「ページ」へ進み、手順1で作ったFacebookページを連携させます。

---

## 第2フェーズ：アプリと「自動化ボット」の作成

> Metaのシステム上で、APIを叩くためのプログラム（アプリ）と、無期限で動いてくれる担当者（ボット）を用意するフェーズです。

### 手順

1. **Metaアプリの作成**
   - [Meta for Developers](https://developers.facebook.com/apps/) で、タイプを「ビジネス」にしてアプリを新規作成し、**アプリID**を控えます。

2. **ビジネスポートフォリオへのアプリ登録**
   - [Metaビジネス設定](https://business.facebook.com/settings) を開き、「アカウント」→「アプリ」から、手順1のアプリIDをリンクさせます。

3. **システムユーザー（API用ボット）の作成**
   - 同じ画面の「ユーザー」→「システムユーザー」から新しいボットを追加します。
   - ⚠️ 役割は必ず **「管理者」** を選択してください。

---

## 第3フェーズ：権限の付与と「最強のパスポート」取得

> 作ったボットに「ページを触っていいよ」という権限を与え、システムに組み込むための無期限アクセストークンを発行するフェーズです。

### 手順

1. **システムユーザーへのアプリ割り当て**
   -Metaビジネス設定の左メニューから 「ユーザー」 ＞ 「システムユーザー」 を開きます。
   -作成したシステムユーザーを選択します。
   -画面右側の 「アセットを追加」 をクリックします。
   -出てきた画面の左側のメニュー（アセットの種類を選ぶところ）で、「アプリ」 を選択します。
   -中央のリストに、先ほど追加したアプリが表示されるので、チェックを入れます。
   -右側の権限一覧で、一番下の 「アプリの管理（フルコントロール）」 のスイッチをオンにします。
   -「変更を保存」 をクリックします。

2. **アセットの割り当て**
   - システムユーザーの画面で「アセットを追加」を押し、対象のFacebookページ（およびInstagramアカウント）の**フルコントロール権限**を付与します。

3. **無期限トークンの生成**
   - 「新しいトークンを生成」を押し、アプリを選択します。
   - 以下の **4つの必須権限** にチェックを入れます：

     | スコープ | 用途 |
     |---|---|
     | `instagram_basic` | 基本情報の読み取り |
     | `instagram_content_publish` | 投稿の公開 |
     | `pages_show_list` | ページ一覧の取得 |
     | `pages_read_engagement` | ページエンゲージメントの読み取り |

   - 発行された長大な文字列（**無期限アクセストークン**）をコピーし、絶対に無くさない安全な場所に保存します。

---

## 第4フェーズ：宛先（Instagram ID）の確認

> 「どこに投稿するか」を指定するための、17桁のIDナンバーを見つけ出すフェーズです。

### 手順

1. **InstagramビジネスアカウントIDの取得**
   - ビジネス設定画面から「アカウント」→「Instagramアカウント」を開きます。
   - ブラウザのURL欄、または画面上に表示されている `178...` から始まる **17桁の数字** をコピーします。

---

## 第5フェーズ：APIへの命令（2ステップ投稿）

> 取得した「トークン」と「ID」をプログラムに組み込んで、実際に画像を投稿させるフェーズです。

### ステップ1：下書き（コンテナ）の作成

```
POST /{17桁のIG_ID}/media
```

| パラメータ | 内容 |
|---|---|
| `image_url` | 画像の公開URL |
| `caption` | 投稿本文 |

✅ APIから **コンテナID（`creation_id`）** が返却されます。

---

### ステップ2：公開（パブリッシュ）

```
POST /{17桁のIG_ID}/media_publish
```

| パラメータ | 内容 |
|---|---|
| `creation_id` | ステップ1で取得したコンテナID |

🎉 これで、Instagramのフィードに実際の画像が投稿されます！
